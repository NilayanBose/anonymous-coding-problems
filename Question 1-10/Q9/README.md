# Problem:</br>
Given a string, find the length of the longest substring without repeating characters.</br>

## Example 1:

### Input: 
"abcabcbb"</br>
### Output: 
3</br>
### Explanation: 
The answer is "abc", with the length of 3.</br>

## Example 2:

### Input: 
"bbbbb"</br>
### Output: 
1</br>
### Explanation: 
The answer is "b", with the length of 1.</br>
